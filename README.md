# NekroAgent 天气查询插件

> 一个基于高德地图 API 的天气查询插件。

## 功能特点

- 查询实时天气
- 查询天气预报
- 支持多个城市
- 简洁的结果展示

## 安装方法

1. 克隆到插件目录
```bash
cd plugins/workdir
git clone https://github.com/YukiAcerium/nekro-plugin-weather.git
```

2. 重启 NekroAgent

## 使用方法

在聊天中让 AI 查询天气：
- "查询北京的天气"
- "上海明天天气怎么样"

## 配置说明

在插件配置中设置：
- `API_KEY`: 高德地图 API Key

## 依赖

- httpx
- nekro-agent
